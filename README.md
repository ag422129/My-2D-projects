A centralized main menu system that connects multiple mini-games built in Unity.

This project acts as a **game launcher**, allowing players to select and switch between different gameplay experiences from one unified menu.

---

## 🎮 Overview

This Main Menu system serves as the entry point for multiple games, including:

- Gold Miner (Physics-based arcade game)
- Space Shooter (Wave-based shooter)
- Tetris (Grid-based puzzle game)
- BreakPong (Arcade hybrid game)

Players can select any game from the menu and instantly start playing.
